## ğŸ¤” How It Works

The application leverages a pure JavaScript/TypeScript templating engine.

1.  **Select & Fill**: The user chooses an NDA type and fills in the required details in a simple, structured form.
2.  **Generate**: On submission, the application merges the user's input with a robust, pre-defined legal boilerplate text.
3.  **Output**: The final, merged document is displayed in a clean text area, ready to be copied or downloaded.

All processing is done **client-side**, meaning none of your sensitive information is ever sent to a server, ensuring complete privacy.
